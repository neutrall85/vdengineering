# Директория для изображений

Поместите следующие изображения в эту директорию:

1. `hero-bg.jpg` - Фоновое изображение для главного экрана (рекомендуемый размер: 1920x1080px)
2. `about.jpg` - Изображение для раздела "О компании" (рекомендуемый размер: 800x600px)
3. `partner-placeholder.png` - Заглушка для логотипов партнёров (рекомендуемый размер: 200x80px)
4. `news-default.jpg` - Изображение по умолчанию для новостей (рекомендуемый размер: 800x600px)

## Источники для бесплатных изображений:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)
